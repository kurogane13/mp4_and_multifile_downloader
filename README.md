  ## üéØ Scope                                                                                                                                                                                                             
                                                                                                                                                                                                                          
  This application is designed to automate the process of downloading media files from forum threads, news articles, and other web pages that may require authentication. It excels at:                                   
                                                                                                                                                                                                                          
  - **Forum Thread Parsing**: Extract media files from multi-page forum discussions                                                                                                                                       
  - **News Article Processing**: Download embedded videos from news websites                                                                                                                                              
  - **Batch Operations**: Process multiple URLs and pages simultaneously                                                                                                                                                  
  - **Authentication Handling**: Support for login-protected content                                                                                                                                                      
  - **Format Flexibility**: Download 280+ different file formats across 6 categories

  ## üîê Authentication                                                                                                                                                                                                    
                                                                                                                                                                                                                          
  ### Credential Storage                                                                                                                                                                                                  
  Credentials are stored in `~/Downloads/download_mp4_credentials_file.txt` with the format:                                                                                                                              
  ```                                                                                                                                                                                                                     
  domain|username|password|description                                                                                                                                                                                    
  ```                                                                                                                                                                                                                     
                                                                                                                                                                                                                          
  ### Security Features                                                                                                                                                                                                   
  - Credentials stored locally (not transmitted)                                                                                                                                                                          
  - Session-based authentication                                                                                                                                                                                          
  - Automatic cookie management                                                                                                                                                                                           
  - Secure password handling                                                                                                                                                                                              
                                                                                                                                                                                                                          
  ### Supported Authentication Types                                                                                                                                                                                      
  - **Form-based login**: Standard username/password forms                                                                                                                                                                
  - **Session cookies**: Maintains login state across requests                                                                                                                                                            
  - **Basic HTTP Auth**: When supported by the target site                                                                                                                                                                
  - **Custom headers**: User-agent and browser simulation                                                                                                                                                                 
                                                                                                                                                                                                                          
  ## üìä Logging System                                                                                                                                                                                                    
                                                                                                                                                                                                                          
  ### Log Categories                                                                                                                                                                                                      
  - **CURL**: HTTP request operations and responses                                                                                                                                                                       
  - **WGET**: File download operations and results                                                                                                                                                                        
  - **ERROR**: Error conditions with detailed context                                                                                                                                                                     
  - **DOWNLOAD**: Download progress and completion status                                                                                                                                                                 
  - **INFO**: General process information                                                                                                                                                                                 
  - **DEBUG**: Detailed debugging information                                                                                                                                                                             
                                                                                                                                                                                                                          
  ### Log File Format                                                                                                                                                                                                     
  ```                                                                                                                                                                                                                     
  [2025-06-10 10:24:18] [CURL] Command: curl -L --user-agent 'Mozilla/5.0...'                                                                                                                                             
  [2025-06-10 10:24:18] [WGET] Command: wget 'https://example.com/file.mp4' -O './downloads/file.mp4'                                                                                                                     
  [2025-06-10 10:24:19] [DOWNLOAD] File: video.mp4, Size: 37M, URL: https://example.com/video.mp4                                                                                                                         
  [2025-06-10 10:24:19] [INFO] MP4 download process completed: 5 pages processed, 12 MP4 files found                                                                                                                      
  ```                                                                                                                                                                                                                     
                                                                                                                                                                                                                          
  ### Log Location                                                                                                                                                                                                        
  - **Directory**: `~/Downloads/mp4_downloader_logs/`                                                                                                                                                                     
  - **Naming**: `domain_authtype_timestamp.log`                                                                                                                                                                           
  - **Example**: `example.com_saved_auth_20250610_102407.log`
